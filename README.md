# Berger

Berger est un démon de triage email open-source écrit en Rust, adossé à
Bichon comme backbone d'archivage. Pipeline déclaratif YAML, filtres
natifs et LLM pluggable (Mistral / Ollama), writeback IMAP en dossiers
pour rester compatible avec tous les clients mail. Tout ce qui sort du
strict tri est délégué en aval via des webhooks.

## Statut
